# Sistema de Gestión de Productos

Aplicación desarrollada en JavaFX con conexión a MySQL para la gestión de productos.

## Descripción

Este sistema permite registrar y visualizar productos almacenados en una base de datos MySQL mediante una interfaz gráfica desarrollada con JavaFX.

## Funcionalidades

- Registrar productos.
- Almacenar información en MySQL.
- Visualizar productos registrados.
- Interfaz gráfica amigable con JavaFX.
- Conexión mediante JDBC.

## Tecnologías Utilizadas

- Java 25
- JavaFX
- MySQL
- JDBC
- Maven
- IntelliJ IDEA
